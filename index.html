SPORT SPOT
Have you ever wanted to see results, reviews and summaries for a sport so you get on google search up Liverpool vs Manchester United results, but you just don’t get all the information you want, so you decide to search Liverpool vs Manchester United highlights. SOCIAL NETWORK API
SOCIAL NETWORK API
This is a social network API that uses MongoDB, Mongoose, Node.js and Express.js to create a database that stores information of users and their thoughts. There are many actions you can do such as "GET ALL USERS", "GET USER BY ID", "CREATE USER", "UPDATE USER", "DELETE USER" and the same thing with thoughts and friends. You can use an app, such as Insomnia, to test these CRUD operations and see how this project works.
